# 🌐 Full-Stack Web Development Practice Projects

This repository contains my personal practice work as I learn Full-Stack Web Development through **Angela Yu’s Web Development Bootcamp (Udemy)** and additional self-driven experimentation.

The goal of this repo is to build strong foundations in both frontend and backend development by working on real projects, APIs, and practical exercises.

---

## 🚀 About This Repository

This is a growing collection of mini-projects and learning exercises covering:

- HTML5 & CSS3 fundamentals
- JavaScript (ES6+ and DOM manipulation)
- Node.js & Express.js
- Working with APIs using Axios
- EJS templating
- Backend routing and middleware
- Full-stack integration concepts

Each folder represents a different exercise, concept, or mini project.

---

### 🔧 Tech Stack Used

- Node.js
- Express.js
- EJS (Embedded JavaScript templates)
- Axios
- HTML/CSS (frontend UI)
- Bored API (external REST API)

- 🔄 Fetch random activity from API

## 📦 Installation & Setup

To run this project locally:

### 1. Clone the repository
```bash
git clone https://github.com/your-username/your-repo-name.git
````

### 2. Navigate into the project folder

```bash
cd your-repo-name
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the server

```bash
node index.js
```

### 5. Open in browser

```
http://localhost:3000
```
